    • O que é

    ADBlock Plus é uma extensão gratuita que permite bloquear anúncios irritantes, desabilitar os domínios de rastreamento e bloqueio
    conhecidos para espalhar malware. Disponível para Android, Chrome, Firefox, Internet Explorer, Maxthon, Opera, Safari e Yandex.
    Adblock Plus usa filtros que você escolher para bloquear todos os elementos indesejados.

    História

    Adblock Plus ( ABP ) é um open-source de filtragem de conteúdo e bloqueio de anúncios de extensão desenvolvido pela Eyeo GmbH (Wladimir Palant), uma empresa de desenvolvimento de software alemã. A extensão foi liberado para Mozilla Firefox (incluindo o Firefox para dispositivos móveis), Google Chrome , Internet Explorer , Microsoft Borda (versão beta), Opera , Safari , Yandex navegador e Android .
    Michael McDonald criou o Adblock Plus 0.5, que melhorou no Adblock original, incorporando as seguintes características:

    • whitelisting;
    • Suporte de bloqueio de imagens de fundo;
    • Subscrição de filtros com um endereço fixo e atualizações automáticas;
    • A capacidade de ocultar elementos HTML , permitindo uma maior variedade de imagens a ser bloqueado;
    • A capacidade de ocultar os anúncios em uma base per-site, em vez de globalmente;
    • Vazamento de memória correções;
    • Melhorias para a interface de usuário.

    McDonald parou o desenvolvimento e transferio para o nome de Wladimir Palant, que lançou Adblock Plus 0.6 com uma base de código reescrito em janeiro de 2006. PC World escolheu Adblock Plus como um dos 100 melhores produtos de 2007.

    Como funciona

    O Adblock Plus não tem nenhuma funcionalidade sozinho. ele irá bloquear o que você adicionar na listas de filtros externos. Como padrão já vem uma lista de filtro. As listas de filtros são uma extensão com um conjunto de regras que dizem ao Adblock Plus que elementos de um site são para bloquear.
    Você pode adicionar qualquer lista de filtros que você deseja. Por exemplo, o bloqueio de rastreamento ou malwares. Você também pode criar suas próprias listas de filtros. Quase todos os filtros são de código aberto, portanto, muitas listas de filtros foram criados por usuários da Internet. As listas de filtros ativados por padrão incluem:

    • Uma lista de bloqueio de anúncios selecionados com base em seu idioma (EasyList)
    • A lista de anúncios Aceitável

    Estes são habilitados para você começar. Você pode removê-los ou adicionar outros. A lista de anúncios Aceitáveis exibe anúncios que cumprem com os critérios de anúncios aceitáveis de ​​acordo pela nossa comunidade de usuários.

    Popup

    O pop-up é uma janela que abre no navegador ao visitar uma página web ou acessar uma hiperligação específica. O pop-up é utilizado pelos criadores do site para abrir alguma informação extra ou como meio de propaganda.
    Algumas empresas começaram a desenvolver softwares e barras de ferramentas, com bloqueio de pop-ups, principalmente sites de busca como o Google, Yahoo e MSN. Há algum tempo, os navegadores já vêm com a opção de bloquear as janelas indesejadas no próprio navegador, como o Firefox, Opera, Netscape, google chrome e Internet Explorer 6 SP2. Mesmo assim os pop-ups continuam usando, em vez de códigos HTML, códigos em Flash e em Javascript, que são muito mais difíceis de serem bloqueados.

    Criar filtro

    Regras básicas de filtro

    O filtro mais trivial que você pode definir, é claro, é o endereço da publicidade notificação a ser bloqueado. No entanto, muitas vezes este endereço muda cada vez que uma página é aberta. Por exemplo, isso poderia ser http://example.com/ads/banner123.gif123 é um número aleatório. Aqui bloqueando o endereço completo vai não ajudá-lo, você precisa de um filtro mais geral - como estar http://example.com/ads/banner*.gif. Ou talvez mesmo http://example.com/ads/*.
    Nota : Certifique-se de que você não está substituindo muito curinga. O filtro http://example.com/*definitivamente bloquear todas as bandeiras, mas tudo o resto ejemplo.com que ainda pode querer a ver também é bloqueado.
